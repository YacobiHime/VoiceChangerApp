## アプリ名：RVC声質変換
RVCモデルを用いた声質変換webアプリケーション。

## 対象者
ゲームや動画のボイスなどで、自分の声を変えてアフレコしたい人。声質変換で遊びたい人。

## アプリの用途
ゲームや動画のボイスなどで、声質を変えてアフレコを行う場合等に使用。

## 使用方法
- 1．Dockerがインストールされている環境を用意し、ターミナルで`Docker compose up --build`を入力する。
- 2．任意のブラウザで`localhost:8501`に接続。
- 3．WAVかMP3のファイルをアップロードするか、マイクで声を収録する。
- 4．モデルを選択し、パラメータを調整する。
- 5．変換を開始して30秒程度待つ。

## アピールポイント
- RVCモデルを扱うために必要なrvc-pythonライブラリのGitHubを確認し、使用方法やパラメータ等を調査した。
- RVCwebUIでは、webサイト上での音声収録をできず、日本訳が分かりづらい。このアプリではそれを改善し、さらに入力データを変換する事に特化させる事で、声質変換機としての使いやすさを向上させた。
